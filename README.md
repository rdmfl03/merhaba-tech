# Merhaba Tech GmbH

Offizielle Website der **Merhaba Tech GmbH** — Entwickler von [MerhabaMap](https://merhabamap.com), der Plattform für türkisch-deutsche Community, Orte und Events.

## Über uns

Merhaba Tech GmbH wurde von **Erdem Oflu** und **Samet Akyazi** gegründet. Wir entwickeln digitale Produkte, die Menschen mit türkischem Bezug in Deutschland verbinden.

## Tech Stack

- [Next.js](https://nextjs.org) 16 (App Router)
- [Tailwind CSS](https://tailwindcss.com) 4
- TypeScript
- Deployment: [Vercel](https://vercel.com)

## Entwicklung

```bash
npm install
npm run dev
```

Die Website läuft unter [http://localhost:3000](http://localhost:3000).

## Design

UI/UX folgt dem MerhabaMap Design-System (Skill: `.cursor/skills/ui-ux-pro-max`):

- Brand-Rot `#E30A17` für primäre Aktionen
- Türkis `#30D5C8` nur für Map/Discovery-CTAs
- Ruhige, vertrauenswürdige Oberflächen
